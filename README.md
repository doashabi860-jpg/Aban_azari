# Trust Wallet Simulator

A premium, realistic Trust Wallet application simulator built with React and TypeScript.

## Features

✅ **Home Page** - Main wallet interface with crypto list
✅ **Receive Page** - Crypto selection and address display
✅ **Realistic UI** - Matches Trust Wallet design exactly
✅ **Interactive Buttons** - Buy, Swap, Receive, Send
✅ **Dark Theme** - Professional matte black interface
✅ **Responsive Design** - Mobile-optimized

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view in your browser.

### Build

```bash
npm run build
```

## Tech Stack

- **React 18** - UI library
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Vite** - Build tool
- **Lucide React** - Icons
